IEEE FIE 2026

Presents a privacy-preserving review pipeline that combines a locally hosted LLM with AST-based static analysis to detect object-oriented design violations in student pull requests without sending code to external services.

  • Evaluated 722 student pull requests from Expertiza across 20 semesters using diff-anchored code context and instructor-graded exemplars for in-context grounding.
  • Assessed a 13-category taxonomy spanning SOLID principles, GRASP patterns, and Fowler code smells to surface recurring issues such as SRP, OCP, CMO, and God Object violations.
  • Showed that several design violations persisted across a decade of submissions, turning those patterns into misconception-targeted teaching recommendations.
